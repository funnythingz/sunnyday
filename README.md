# Sunnyday

Sunnyday is a Truncate implemented in Go.

## Installation

```
% go get github.com/funnythingz/sunnyday
```

## Usage

```
package main

import(
  "fmt"
  "github.com/funnythingz/sunnyday"
)

func main() {
    fmt.Println(Truncate("うんこっこたのしいお！", 5))
}
```
